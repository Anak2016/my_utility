=====================
== about example_code folder
====================
> example_code folder contains example of code that is useful for reader to quickly recall usecases of
    methods/functions/classes that are implemented.

=====================
== Rules
====================
> usecase of example_code must be clear and unambigious.
> code must includes clear comments
> in the doc of function/classes/methods, url + article's name + published time must be on the first line.
    : nothing else should be on the first line.
